1. In jQuery terminology, what is the difference between the “jQuery function” and the “jQuery object”? 

==> jQuery function is: a set of statements that takes input, do some specific computation and produce output.
or that performs some specific task or does some computation and then return the result to the user.
==> A jQuery object is: an object returned by the jQuery function. A jQuery object represents a set of document elements 
and can also be called a “jQuery result”, a “jQuery set”, or a “wrapped set”. 


2. Write jQuery code to find all h1 elements that are children of a div element and make their background 
color red.
Sample HTML:
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>

<body>
    <h1>abc</h1><br><div>
        <h1>div-1</h1>
        <h1>div-2</h1>
        </div>
        <h1>xyz</h1>

        <script>
        $(document).ready(function(){
           $("div h1").css("backgroundColor","red");
       });
    </script>  
</body>
